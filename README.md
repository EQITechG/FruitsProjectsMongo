### Installation


1. Get a free MongoDB Key at <a href="https://www.mongodb.com/">MongoDB</a>

2. Install Nodemon
    ```sh
   npm install -g nodemon
   ```
3. Initiate NPM 
    ```sh
   npm init 
   ```

4. Install NPM packages
    ```sh 
    npm i body-parser ejs express lodash mongose request
  
5. Run app
    ```sh
   nodemon app.js 
   ```
6. Use app by visiting 
     ```sh
   localhost:3000 
   ```
